Cary: Car Management & Analytics App 🚗
Cary is a premium, subscription-based mobile application designed to help car owners manage their vehicles, track maintenance schedules, and monitor fuel expenses with ease.

🚀 Key Features
Expense Tracking: Monitor fuel consumption and maintenance costs.

Maintenance Schedules: Stay on top of oil changes, tire rotations, and service appointments with smart reminders.

Fuel Analytics: Visualize your vehicle's fuel efficiency over time.

Premium UI: A modern, dark-themed interface built for a seamless user experience.

🛠 Tech Stack
Framework: Flutter

Language: Dart

State Management: BLoC (Cubit)

Backend: Firebase (Authentication & Database)

Monetization: RevenueCat for subscription services

🏗 Project Architecture
The project follows a clean architecture pattern to ensure maintainability and scalability, using the BLoC pattern for efficient state management.
